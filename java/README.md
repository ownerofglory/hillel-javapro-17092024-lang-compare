# Hillel Java Pro: Example in Java

## Project structure
```sh
.
├── App.java               # Main class
├── README.md
└── mathlib                # Math package
    └── MathUtil.java      # Math lib class
```

## How to build and run
- Compile Java Sources
```sh
    javac App.java
```
- Run on JVM
```sh
    java App
```
